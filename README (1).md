# A Realty Group CRM

Система управления недвижимостью с интеграцией Bitrix24

## 🚀 Быстрый старт

1. Создайте новый репозиторий на GitHub
2. Загрузите файл `index.html` в репозиторий
3. Включите GitHub Pages в настройках репозитория
4. Готово! Сайт будет доступен по адресу `https://ваш-username.github.io/репозиторий/`

## ⚙️ Настройка Bitrix24

### 1. Вебхук

В коде уже настроен вебхук:
```javascript
const BITRIX_WEBHOOK = 'https://b24-6mcve5.bitrix24.kz/rest/1/0bcsjte5yrhzozk4/';
```

Если нужен свой вебхук:
1. В Bitrix24: **Настройки → Интеграции → Входящие вебхуки**
2. Создайте вебхук с правами на CRM
3. Замените URL в коде

### 2. Категория сделок

Объекты загружаются из категории ID = 8 (Алатауский Район)

```javascript
const CATEGORY_ID = 8;
```

## 📸 Добавление фото объектов

### Способ 1: Через вложения Bitrix24

1. Откройте сделку в Bitrix24
2. Перейдите в раздел **"Файлы"**
3. Загрузите фото объекта
4. Фото автоматически появится в CRM

### Способ 2: Через UF поле

1. Создайте пользовательское поле в Bitrix24:
   - **Настройки → Настройки CRM → Пользовательские поля**
   - Тип: **Файл** или **Строка** (для URL)
   - Название: "Фото объекта"

2. Узнайте код поля:
   ```javascript
   // В консоли браузера на странице Bitrix24
   BX24.callMethod('crm.deal.fields', {}, console.log)
   ```

3. Обновите код в `index.html`:
   ```javascript
   // Найдите эту строку и замените на свой код поля
   &select[]=UF_CRM_1709123456&select[]=UF_CRM_1709123457
   ```

## 📱 Скрытый номер продавца

### Настройка UF поля для телефона

1. Создайте поле в Bitrix24:
   - Тип: **Строка**
   - Название: "Телефон продавца"

2. Узнайте код поля (например: `UF_CRM_1709123456`)

3. Обновите код:
   ```javascript
   // Найдите в index.html:
   const sellerPhone = deal.UF_CRM_1709123456 || deal.UF_CRM_1709123457 || '';
   
   // Замените на свой код поля
   const sellerPhone = deal.ВАШ_КОД_ПОЛЯ || '';
   ```

Номер будет отображаться как: `••• ••• •• 45` (последние 2 цифры видны)

## 🤖 Telegram уведомления

1. Создайте бота через [@BotFather](https://t.me/BotFather)
2. Получите токен бота
3. Создайте группу в Telegram и добавьте бота
4. Получите `chat_id` группы
5. Обновите код:
   ```javascript
   const TELEGRAM_CONFIG = {
       botToken: 'ВАШ_ТОКЕН',
       chatId: 'ВАШ_CHAT_ID',
       enabled: true
   };
   ```

## 📞 Интеграция Mango (опционально)

Для переадресации звонков и записи:

1. Получите API ключ в [личном кабинете Mango](https://lk.mango-office.ru/)
2. Обновите конфигурацию:
   ```javascript
   const MANGO_CONFIG = {
       apiKey: 'ВАШ_API_KEY',
       apiSalt: 'ВАШ_API_SALT',
       enabled: true
   };
   ```

## 🔄 Обновление с GitHub

После загрузки на GitHub:

1. Перейдите в **Settings** репозитория
2. В разделе **Pages** выберите:
   - Source: **Deploy from a branch**
   - Branch: **main** / **master**
   - Folder: **/(root)**
3. Нажмите **Save**
4. Через 1-2 минуты сайт будет доступен

## 🛠️ Кастомизация

### Изменение процента комиссии

Найдите в коде:
```javascript
<input type="number" id="commissionPercent" value="2" oninput="calculateCommission()">
```

Замените `value="2"` на нужный процент.

### Изменение цветов

Цвета настраиваются в CSS секции:
```css
/* Основной градиент */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* Статусы */
.status-new { background: #e3f2fd; color: #1976d2; }
.status-work { background: #fff3e0; color: #f57c00; }
```
